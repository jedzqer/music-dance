# 音乐可视化

一个浏览器端音乐可视化播放器。支持选择或拖放本地音频文件播放，根据音频频谱绘制动态 Canvas 视觉效果，并展示播放控制、进度条、音量控制、专辑封面、内嵌歌词和不同播放器布局样式。

## 功能

- 本地音频文件选择与拖放播放
- Canvas 音乐频谱与粒子动效可视化
- 播放 / 暂停、音量调节、进度拖动
- 读取音频内嵌封面并提取主题色
- 读取并展示内嵌歌词，支持 LRC 时间轴高亮
- 两种播放器布局样式切换

## 运行方法

先安装依赖：

```bash
npm install
```

启动本地服务：

```bash
npm start
```

然后根据终端输出的地址在浏览器中打开页面。
